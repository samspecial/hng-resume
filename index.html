Turnbase Game

Frontend

This is a two-player turnbase game that depicts the medieval battle of the period between 500 - 1500AD

The features of the game are obstacles represented with bricks and weapons represented with arrow, gun, bow and grenade.

Movement - Players can make a move between 1 - 3 square along the four cardinal direction (up, right, down, left). Player can not move over an obstacle (brick)

Picking Weapon - Players can pick a weapon by landing on the square the weapon appears. This weapon is set as current player weapon and any former weapon is dropped immediately the player moves out of the square

Fight - A supremacy battle is activated once both players are within 1 - 3 squares apart along the same direction (horizontal or vertical).

This game was built using barebone JavaScript and jQuery for DOM traversal.
